# NGANYA

NGANYA is an Android application inspired by Instagram and TikTok with a cool theme and iconist design for entertainment.

## Project Details

- **Package Name**: com.nganya.app
- **Language**: Kotlin
- **Minimum SDK**: API 24 (Android 7.0)
- **Target SDK**: API 34+

## Getting Started

### Prerequisites

- Android SDK Platform
- Android SDK Build-Tools
- Android Emulator (optional)
- Kotlin 1.9+

### Installation

1. Clone the repository
```bash
git clone https://github.com/illegalboyillegalpablo-cell/Nyanya.git
cd Nyanya
```

2. Build the project
```bash
./gradlew build
```

3. Run on emulator or device
```bash
./gradlew installDebug
```
